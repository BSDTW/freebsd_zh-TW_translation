19.5 哪裡可以找到兼容 ANSI 的 lint？

PC-Lint 和 FlexeLint 是 Gimpel Software 公司的 產品 (http://www.gimpel.com/)。

Unix System V 版本 4 的 lint 兼容 ANSI。 可以從 UNIX Support Labs 或 System V 的銷售商 單獨得到 (和其它 C 工具捆綁在一起)。

另外一個兼容 ANSI 的 lint 是 Splint (以前叫 lclint, http://www.splint.org/)。它可以 作一些高級別的正式檢驗。

如果沒有 lint, 許多現代的編譯器可以作出幾乎和 lint 一樣多的診斷。許多網友推薦 gcc -Wall -pedantic。
