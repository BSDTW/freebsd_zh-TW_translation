19.1 常用工具列表。

工具 程序名 (參見問題 18.18)
C 交叉引用生成器 cflow, cxref, calls, cscope, xscope, ixfw
C 源代碼美化器/美化打印 cb, indent, GNU indent, vgrind
版本控制和管理工具 CVS, RCS, SCCS
C 源代碼擾亂器 (遮蔽器) obfus, shroud, opqcp
``make" 從屬關係生成器 makedepend, cc -M 或 cpp -M
計算源代碼度規工具 ccount, Metre, lcount, csize; McCable and Associates 也有一個商業包出售
C 源代碼行數計數器 可以用 UNIX 的標準工具 wc 作個大概的計算, 比用 grep -c ";" 要好
C 說明幫助 (cdecl) 見 comp.sources.unix 第14卷  (參見問題 18.18) 和 [K&R2]
原型發生器 參見問題 11.30
malloc 問題抓捕工具 參見問題 18.2
``選擇性" 的 C 預處理器 參見問題 10.16
語言翻譯工具 參見問題 11.30 和 20.23
C 校對機 (lint) 參見問題 18.5
C 編譯器 參見問題 18.3

這個工具列表並不完全, 如果你知道有沒列出的工具, 歡迎聯繫本表的維護者。

其它工具列表和關於它們的討論可以在 Usenet 的新聞組  comp.compilers 和 comp.software-eng 找到。
