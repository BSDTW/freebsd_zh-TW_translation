20.11 怎樣做圖形？

從前, Unix 下有一套相當不錯且小巧的設備獨立的繪製函數 (plot(3)  和 plot(5))。由 Robert Maier 寫的 GNU libplot 函數庫 保持了同樣的精神, 支持許多現代的繪製設備  (http://www.gnu.org/software/plotutils/plotutils.html)。

OpenGL 是一個現代的平台獨立的製圖函數庫, 它也支持三維製圖和動畫。 其它有關的製圖標準有 GKS 和 PHIGS。

如果你在 MS-DOS 下編程, 你大概需要用到符合 VESA 或 BGI  標準的函數庫。

如果你要和某一個特定的製圖儀打交道, 通常發送適當的逃逸序列就可以繪圖了, 參見問題 19.9. 銷售商可能會提供一個可以從 C 調用的函數庫, 或者 你也許可以在網上找到。

如果你需要在某個特定的視窗環境下編程 (Macintosh, X Window, Microsoft Windows), 你使用其提供的工具; 參閱相關的文檔、新聞組 或 FAQ。

參考資料: [PCS, Sec. 5.4 pp. 75-77]。
