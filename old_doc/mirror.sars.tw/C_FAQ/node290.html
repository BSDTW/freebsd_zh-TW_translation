19.3 有什麼免費或便宜的編譯器可以使用？

自由軟件基金的 GNU C (gcc, http://gcc.gnu.org/) 是個流行而高質量的 免費 C 編譯器. djgpp (http://www.delorie.com/djgpp/) 是移植到  MS-DOS 的 GCC 版本。據我所知, 有移植到 Macs 和 Windwos 上 的 GCC 版本。 19.1

lcc 是另外一個流行的編譯器  http://www.cs.virginia.edu/~lcc-win32/,  http://www.cs.princeton.edu/software/lcc/）。

Power C 是 Mix Sotfware 公司提供的一個非常便宜的  MS-DOS 下的編譯器。公司地址：1132 Commerce Drive, Richardson, TX 75801, USA, 214-783-6001。

ftp://ftp.hitech.com.au/hitech/pacific 是個 MS-DOS 下的試用  C 編譯器。非商業用途的不一定要註冊。

新聞組 comp.compilers 的檔案中有許多有關各種語言的編譯器、 解釋器、 語法規則的信息。 新聞組在 http://compilers.iecc.com/ 的 檔案包含了一個 FAQ 列表和免費編譯器的目錄。
