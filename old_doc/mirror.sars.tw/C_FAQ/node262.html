16.5 怎樣寫類似 printf() 的函數, 再把參數轉傳給 printf() 去 完成大部分工作？

用 vprintf(), vfprintf() 或 vsprintf()。

下面是一個 error() 函數, 它列印一個出錯信息, 在信息前加入字符串  ``error: " 和在信息後加入換行符： 

        #include <stdio.h>
        #include <stdarg.h>

        void error(const char *fmt, ...)
        {
	    va_list argp;
	    fprintf(stderr, "error: ");
	    va_start(argp, fmt);
	    vfprintf(stderr, fmt, argp);
	    va_end(argp);
	    fprintf(stderr, "\n");
        }
