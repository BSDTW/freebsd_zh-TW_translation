19.9 有什麼好的學習 C 的書？有哪些高級的書和參考？

有無數有關 C 的書, 我們無法一一列出, 也無法評估所有的書。 許多人相信最好的書, 也是第一本：由 Kernighan 和 Richie  編寫的《The C programming Language》(``K&R", 現在是 第二版了 [K&R2])。對這本書是否適合初學者有不同的意見; 我們當中許多人 是從這本書學的 C, 而且學的不錯; 然而, 有些人覺得這本書太客觀了點, 不大適合那些沒有太多程序設計經驗的人作為第一個教程。網上有一些 評注和勘誤表, 例如:  http://www.csd.uwo.ca/~jamie/.Refs/.Footnotes/C-annotes.html,  http://www.eskimo.com/~scs/cclass/cclass.html,  http://cm.bell-labs.com/cm/cs/cbook/2ediffs.html。

許多活躍在新聞組 comp.lang.c 的人推薦 K.N. King 寫的 《C: A Modern Approach》。

一本極好的參考書是由 Samuel P. Harbison 和 Guy L. Steele 和寫的 《C: A Reference Manual》[H&S]。

雖然並不適合從頭開始學 C, 本 FAQ (英文版)已經出版了, 見文獻 [CFAQ]。

C 和 C++ 用戶協會 (Association of C and C++, ACCU) 維護著一份 很全面的有關 C/C++ 的書目評論 (http://www.accu.org/bookreviews/public/)。
