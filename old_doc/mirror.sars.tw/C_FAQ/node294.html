19.7 網上有哪些 C 的教程或其它資源？

有許多個:

在 http://cprog.tomsweb.net 有個 Tom Torfs 的不錯的教程。

Christopher Sawtell 寫的《給 C 程序員的便筏》 (Notes for C programmers)。 在下面的地址可以得到:  ftp://svr-ftp.eng.cam.ac.uk/misc/sawtell_C.shar,  ftp://garbo.uwasa.fi/pc/c-lang/c-lesson.zip,  http://www.fi.uib.no/Fysisk/Teori/KURS/OTHER/newzealand.html。

Time Love 的《程序員的 C》 (C for Programmers)。  http://www-h.eng.cam.ac.uk/help/tpl/languages/C/teaching_C/

The Coronado Enterprises C 教程在 Simtel 鏡像點 目錄 pub/msdos/c, 或在 http://www.coronadoenterprises.com/tutorials/c/index.html。

Steve Holmes 的在線教程 http://www.strath.ac.uk/IT/Docs/Ccourse/。

Martin Brown 的網頁有一些 C 教程的資料  http://www-isis.ecs.soton.ac.uk/computing/c/Welcome.html。

在一些 UNIX 的機器上, 在 shell 命令行, 可以試試 ``learn c"。注意教程可能比較舊了。

最後, 本 FAQ 的作者以前教授一些 C 的課程, 這些筆記 都放在了網上  http://www.eskimo.com/~scs/cclass/cclass.html。

【不承諾申明: 我沒有檢閱這些我收集的教程, 它們可能含有錯誤。除了那個 有我名字的教程, 我不能為它們提供保證。而這些信息會很快的變得過時, 也許, 當你讀到這想用上面的地址時, 它們已經不能用了】

這其中的幾個教程, 再加上許多其它 C 的信息, 可以從  http://www.lysator.liu.se/c/index.html 得到。

Vinit Carpenter 維護著一個學習 C 和 C++ 的資源列表, 公佈在新聞組  comp.lang.c 和 comp.lang.c++, 也歸檔在本 FAQ 所在  (參見問題 20.36), 或者  http://www.cyberdiem.com/vin/learn.html。

參見問題 18.8、18.9 和 18.16。
