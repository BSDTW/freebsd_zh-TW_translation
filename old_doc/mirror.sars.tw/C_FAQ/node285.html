18.7 哪裡可以找到 ``印第安山風格指南" (Indian Hill Style Guide) 及其它編碼標準？

各種文檔在匿名 ftp 都可以得到：

地址 文檔及目錄
ftp.cs.washington.edu pub/cstyle.tar.Z
  (更新的印第安山風格指南, (Indian Hill Guide))
ftp.cs.toronto.edu doc/programming
  (包括 Henry Spencer 的《C 程序員的十誡》 (``10 Commandments for C Programmers"))
ftp.cs.umd.edu pub/style-guide

也許你會對這些書感興趣: 《The Elements of Programming Style》[K&P], 《Plum Hall Programming Guidelines》[Plum], 《C Style: Standards and Guidelines》[Straker]。參見文獻 [21]。
