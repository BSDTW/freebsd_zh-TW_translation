19.15 誰有 C 編譯器的測試套件？

Plum Hall (以前在 Cardiff, NJ, 現在在 Hawaii) 有一個套件出售; Ronald Guilmette 的 RoadTestTM 編譯器測試套件 (更多 信息在  ftp://netcom.com/pub/rfg/roadtest/announce.txt);  Nullstone 的自動編譯器性能分析工具 (http://www.nullstone.com)。  FSF 的 GNU C (gcc) 發佈中含有一個許多編譯器通常問題的 C 嚴酷測試。  Kahan 的偏執狂的測試 (ftp://netlib.att.com/netlib/paranoia), 盡其所能的測試 C 實現的浮點能力。
