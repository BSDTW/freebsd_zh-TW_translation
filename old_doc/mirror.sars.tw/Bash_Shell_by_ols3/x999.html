無窮迴圈

第一種做法

while :
do
    r=$(( $RANDOM % 42 ))
    r=$[ r + 1]
    echo $r
    sleep 2 
done

註:  ':' 是表示什麼都不做之意

第二種：

while true
do
	echo "Hi"
done

或

while ! false
do
	echo "Hi"
done

註: 
true 是一命令列程式，用來傳回 0 值(即為真)
false 則用來傳回非 0 值(即為假)


第三種：

until false
do
	echo "Hi"
done


第四種：

for ((;;))
do
   echo "Hi"
   sleep 2
done


第五種：

for ((n=1; n>0; n++))
do
   echo "$n"
   sleep 2
done
