17.select 選單語法

select 可方便製作一個簡易的選單。

select 的寫法

#! /bin/sh

select fn in *; do
    # 若不在選項中, 則離去
    if [ -z $fn ]; then
	break
    fi
    echo "$fn (you choose $REPLY)"
done

# 若 select 及 done 中間沒有 break, 則會不斷地 loop
# 選項數字被存入 $REPLY 中

=======================
