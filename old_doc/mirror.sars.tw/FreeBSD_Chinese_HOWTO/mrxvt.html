7.17. multi-aterm

安裝 x11/multi-aterm：

# cd /usr/ports/x11/multi-aterm
# make CONFIGURE_ARGS="--enable-big5" install clean

Note: 不幸的是 mrxvt 0.2.1 並不支援中文， mrxvt 0.3.11 則用 make CONFIGURE_ARGS="--enable-cjk --enable-xim --enable-xft" install clean
