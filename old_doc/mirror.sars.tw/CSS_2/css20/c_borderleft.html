border-left版本：CSS1　 兼容性：IE4+ NS6+　繼承性：無
語法：
border-left : border-width || border-style || border-color
取值：
該屬性是復合屬性。請參閱各參數對應的屬性。
默認值為： medium none 。 border-color 的默認值將採用文本顏色。
說明：
設置對像左邊框的樣式。
當你指定了邊框顏色( border-left-color )和邊框寬度( border-left-width )時，你必須同時指定邊框樣式( border-left-style )，否則邊框不會被呈現。
如使用該復合屬性定義其單個參數，則其他參數的默認值將無條件覆蓋各自對應的單個屬性設置。
例如：設置 border-left : thin 等於設置 border-left : thin none ，而 border-left-color 的默認值將採用文本顏色。因此此前的任何 border-left-color 和 border-left-width 設置都會被清除。
在 IE5.5+ 中，邊框屬性可以直接應用於內聯要素。而在此前的版本中，內聯要素要使用該屬性，必須先設定對象的 height 或 width 屬性，或者設定 position 屬性為 absolute ，或者設定 display 屬性為 block 。
對應的腳本特性為 borderLeft 。
示例：
div { border-bottom: 25px solid red; border-left: 25px solid yellow; border-right: 25px solid blue; border-top: 25px solid green; }
