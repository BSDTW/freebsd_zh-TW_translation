Attribute Selectors　版本：CSS2　 兼容性：無
語法：
  1. E [ attr ] { sRules }
  2. E [ attr = value ] { sRules }
  3. E [ attr ~= value ] { sRules }
  4. E [ attr |= value ] { sRules }
說明：
屬性選擇符。
  1. 選擇具有 attr 屬性的 E
  2. 選擇具有 attr 屬性且屬性值等於 value 的 E
  3. 選擇具有 attr 屬性且屬性值為一用空格分隔的字詞列表，其中一個等於 value 的 E 。這裡的 value 不能包含空格
  4. 選擇具有 attr 屬性且屬性值為一用連字符分隔的字詞列表，由 value 開始的 E
示例：
h[title] { color: blue; }
/* 所有具有title屬性的h對像 */
 
span[class=demo] { color: red; }
 
div[speed="fast"][dorun="no"] { color: red; }
 
a[rel~="copyright"] { color:black; }
