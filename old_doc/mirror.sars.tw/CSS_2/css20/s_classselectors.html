Class Selectors　版本：CSS1　 兼容性：IE4+ NS4+
語法：
E.className { sRules }
說明：
類選擇符。在HTML中可以使用此種選擇符。其效果等同於E [ class ~= className ] 。請參閱屬性選擇符( Attribute Selectors )。
在IE5+，可以為對象的 class 屬性(特性)指定多於一個值( className )，其方法是指定用空格隔開的一組樣式表的類名。例如：<div class="class1 class2">。
示例：
div.note { font-size:14px; }
/* 所有class屬性值等於（包含）"note"的div對像字體尺寸為14px */
 
.dream { font-size:14px; }
/* 所有class屬性值等於（包含）"note"的對象字體尺寸為14px */
