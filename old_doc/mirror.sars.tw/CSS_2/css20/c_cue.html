cue-before版本：CSS2　 兼容性：無　繼承性：無
語法：
cue : cue-before || cue-after
取值：
該屬性是復合屬性。請參閱各參數對應的屬性。
說明：
設置在對像前後播放的聲音。
假如只給出了一個值，它將用於 cue-before 屬性和 cue-after 屬性。假如給出了兩個值，則第一個用於 cue-before 屬性，第二個用於 cue-after 屬性。
對應的腳本特性為 cue 。
示例：
div { cue: url(orchestralsneeze.wav) none }
div { cue: url(orchestralsneeze.wav); }
