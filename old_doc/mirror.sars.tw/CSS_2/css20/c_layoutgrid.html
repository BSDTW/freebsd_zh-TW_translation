layout-grid版本：IE5+專有屬性　繼承性：有
語法：
layout-grid : layout-grid-mode || layout-grid-type || layout-grid-line || layout-grid-char
取值：
該屬性是復合屬性。請參閱各參數對應的屬性。
默認值為：both loose none none。
說明：
設置或檢索復合文檔中指定文本字符版式的網格特性。
以東亞語言(如中文，日文)編寫的文檔通常根據一維或二維網格使用字符的頁面版式。
對應的腳本特性為 layoutGrid 。
示例：
div { layout-grid:both loose none none; }
