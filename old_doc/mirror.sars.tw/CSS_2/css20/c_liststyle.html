list-style版本：CSS1　 兼容性：IE4+ NS4+　繼承性：有
語法：
list-style : list-style-image || list-style-position || list-style-type
取值：
該屬性是復合屬性。請參閱各參數對應的屬性。
默認值為：disc outside none。
說明：
設置列表項目相關樣式。
當 list-style-image 和 list-style-type 都被指定了時， list-style-image 將獲得優先權。除非 list-style-image 設置為 none 或指定 url 地址的圖片不能被顯示。
對應的腳本特性為 listStyle 。
示例：
li { list-style: url("http://www.dhtmlet.com/devgurupix.gif") inside circle; }
ul { list-style: outside upper-roman; }
ol { list-style: square; }
