:first-letter版本：CSS2　 兼容性：IE5.5+
語法：
Selector : first-letter { sRules }
說明：
設置對像內的第一個字符的樣式。
此偽對像僅作用於塊對象。內聯要素要使用該屬性，必須先設定對象的 height 或 width 屬性，或者設定 position 屬性為 absolute ，或者設定 display 屬性為 block 。

在此偽對像中配合使用 font-size 屬性和 float 屬性可以製作首字下沉效果。
示例：
p a:first-letter { color: green }
div:first-letter { color:red;font-size:16px;float:left; }
