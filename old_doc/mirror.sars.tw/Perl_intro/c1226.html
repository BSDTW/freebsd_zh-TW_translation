21. 找尋大檔案

#! /usr/bin/perl

use File::Find;

# -s 會檢查檔案大小是否為 0，並且傳回 file size
# 本程式利用 -s 的特性，一面尋找大於 1MB 的檔案，也一面秀出其檔案大小。

find( sub { print "$_ -> $filesize\n" if ($filesize=-s $_) > 1_024_000; } , "/home/ols3" );
