4.2.10 將一個陣列逆轉

reverse 可以將一個陣列以相反的順序排列，並傳回排列的結果，但原陣列不受影響，如下所示：

#! /usr/bin/perl

@total=(2, 4, 6, 1, 3, 5, "a", "b", "c");

@xyz= reverse @total;

print @xyz;
# 此時 @xyz 應為 ("c", "b", "a", 5, 3, 1, 6, 4, 2);
# 但 @total 不受影響，仍為 (2, 4, 6, 1, 3, 5, "a", "b", "c");
