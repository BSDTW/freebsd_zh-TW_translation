7.3 叫用副程式

方法 1：

	$rt = max(20, 30);

方法 2：

	$rt = &max(20, 30);

用例：

#! /usr/bin/perl

print "請輸入第一個數：";
chomp($x=<STDIN>);

print "請輸入第二個數：";
chomp($y=<STDIN>);

$max_num_is = max($x, $y);

print "$x 和 $y 中，較大者為： $max_num_is\n\n";


sub max {
	my ($a, $b) = @_; # 取得傳入的二個引數

	($a > $b) ? $a : $b; # 傳回較大值
}

註：副程式可以放在程式中的任何一個位置。
