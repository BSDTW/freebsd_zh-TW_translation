4.1.12 比較算符

純量除了運算之外，還有一種比較算符，用來比較大小及順序。

Perl 的比較算符有二套，數值用一套，字串用一套。記住這個特色，有利於您快速掌握 Perl !!

數值的比較算符：

大於   >
等於   ==
小於   <
不等於 !=
大於或等於 >=
小於或等於 <=


字串的比較算符：

大於   gt
等於   eq
小於   lt
不等於 ne
大於或等於 ge
小於或等於 le

要特別注意的是：字串之間比較大小，是將二個字串的字元，逐一以字元在 ASCII 表中順序，做為比較大小的依據：

$a = "abcde";
$b = "ABCDE";

if ($a gt $b) {
   print '$a 大於 $b';
} elsif ($a eq $b) {
   print  '$a 等於 $b';
} else {
   print  '$a 小於 $b';
}

結果：

$a 大於 $b

(因為 ASCII 中，大寫英文字母排在小寫字母之前)
