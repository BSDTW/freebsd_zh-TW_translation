5.3 until 控制結構

until 是指直到某事為真之前(若它為假)，都一直做某事。

until 有二種形式：

第 1 種：

until (條件判斷為假) {

	一直做某事;

}

例：

until ($sun eq 'is down') {

	print "現在是白天 ....\n";
}



第 2 種：(後置式)

一直做某事 until (條件判斷為假);


例：

$i++ until ($i > 1000);



第 3 種：

do {
	一直做某事 

} until (條件判斷為假);

例：

#! /usr/bin/perl

do {

	print "$j\n";
	$j++;

} until ($j>=10);
