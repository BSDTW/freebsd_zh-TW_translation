4.2.3 走過每一個陣列元素

這個意思是說，要將某一陣列的每一個元素取出來看看。列舉以下二種作法：

利用 for 

#! /usr/bin/perl

@total=(2, 4, 6, 1, 3, 5, "a", "b", "c");

# 陣列 index 由 0 開始，一直到最後一元素的 index：$#total (即 8)
for ($i=0; $i<=$#total; $i++) {
	$j=$i+1;
	print "第 $j 個元素是：$total[$i]\n"; 
}

利用 foreach 

#! /usr/bin/perl

@total=(2, 4, 6, 1, 3, 5, "a", "b", "c");

# 陣列 index 由 0 開始，一直到最後一元素的 index：$#total (即 8)
$j=1;
foreach $item (@total) {
	print "第 $j 個元素是：$item\n"; 
	$j++;
}
