5.4 for 控制結構

for 是迴圈的一種，經常被使用在固定迴圈次數時(但不一定，也可能產生無窮迴圈)。

#! /usr/bin/perl

for ($i=1; $i<=100; $i++) {
	$sum += $i;
}

print "總和：$sum\n";

特別注意!!!! foreach 和 for 是可以互用的，也就是說以下例子，結果同上。

#! /usr/bin/perl

# 使用 foreach 
foreach ($i=1; $i<=100; $i++) {
	$sum += $i;
}

print "總和：$sum\n";
