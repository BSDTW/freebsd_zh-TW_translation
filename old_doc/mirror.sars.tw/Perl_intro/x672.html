5.5 while 控制結構

while 是迴圈的一種，經常被使用在某些資料存在時：

第 1 種：

while ( ($k, $v) = each %HASH ) {
	print "$k ===> $v\n";
}


第 2 種：
do {

	$i++;

	print $i, "\n";

} while ( $i < 100 );



第 3 種：使用 continue block 

$i=1;
while ($i < 10) {

	print $i, "\n";

} continue {

	$i++;
}
