4.1.14 undef 的應用

undef 是累加時常用的技巧，下列中，$sum 第一次值為 undef，往後逐次增加。

#! /usr/bin/perl

$i=1;

while ($i < 50) {
	$sum += $i; # 同於 $sum = $sum + $i;
	$i++;       # 同於 $i = $i + 1;
}
print "累加總和是：$sum\n";

結果：

累加總和是：1225


註：

$sum -= $i;  同於  $sum = $sum - $i;

$sum *= $i;  同於  $sum = $sum * $i;

$sum /= $i;  同於  $sum = $sum / $i;

undef 可將變數還原成未定義狀態：

#! /usr/bin/perl

$i=1

while ($i < 50) {
	$sum += $i;
	$i++;
}

$sum=undef;

print "累加總和是：$sum\n";

結果：

累加總和是：
