4.1.10 數值和字串也可以運算

數值和字串是可以一起運算的，結果視其運算目的而定。

$v1 = 99;
$v2 = '121';

print $v1 + $v2;   (欲計算數值)

結果：220 (數值)


$v1 = 99;
$v2 = '121';

print $v2 . $v1;   (欲形成字串)

結果：12199 (字串)


$v1 = '324'; (字串)

print $v1 * 1;     (欲計算數值)

結果：324 (數值)

雖然數值和字串可以混合運算，但若太奇怪的組合，便失去其用處。

比如：

$v1 = "jack die\n";

$v2 = 20;

print $v1 * $v2;

試問，這有何意義可言？因此，適度的運用，有益；過度，則造成意義混淆不明。
