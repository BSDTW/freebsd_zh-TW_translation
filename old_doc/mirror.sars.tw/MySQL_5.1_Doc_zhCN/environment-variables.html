附錄F：環境變數

在本附錄中，列出了MySQL直接或間接使用的環境變數。大多數環境變數均能在本手冊的其他部分找到。

注意，命令行上的任何選項均優先於選項檔案和環境變數中指定的值，而且選項檔案中的值優先於環境變數中的值。。

在很多情況下，要想更改MySQL的行為，應優先使用選項檔案而不是環境變數。請參見4.3.2節，「使用選項檔案」。

變數

描述

CXX

C++編譯器的名稱（用於運行configure）

CC

C編譯器的名稱（用於運行configure）

CFLAGS

C編譯器的標誌（用於運行configure）

CXXFLAGS

C++編譯器的標誌（用於運行configure）

DBI_USER

Perl DBI的預設帳號

DBI_TRACE

用於Perl DBI的跟蹤選項

HOME

mysql歷史檔案的預設路徑是$HOME/.mysql_history。

LD_RUN_PATH

用於指定libmysqlclient.so的位置。

MYSQL_DEBUG

調試時的調試跟蹤選項。

MYSQL_HISTFILE

mysql歷史檔案的路徑。如果設置了該變數，它的值將覆蓋$HOME/.mysql_history的預設值。

MYSQL_HOST

mysql命令行客戶端使用的預設主機名。

MYSQL_PS1

在mysql命令行客戶端中使用的命令提示。

MYSQL_PWD

連接到mysqld時的預設密碼。注意，使用它並不安全。請參見5.8.6節，「使您的密碼安全」。

MYSQL_TCP_PORT

預設的TCP/IP端口號

MYSQL_UNIX_PORT

預設的Unix套接字檔案名，用於連接到本地主機。

PATH

由shell使用以找到MySQL程式。

TMPDIR

建立臨時檔案的目錄。

TZ

用於設置當地時區。請參見A.4.6節，「時區問題」。

UMASK_DIR

建立目錄時的用戶目錄建立掩碼。注意，這是帶有UMASK的ANDed。

UMASK

建立檔案時的用戶檔案建立掩碼。

USER

在Windows和NetWare上連接到mysqld時使用的預設帳號。
