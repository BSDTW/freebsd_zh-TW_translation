Debian 參考手冊

摘要

Debian 參考手冊 的目標在為 Debian 的使用者提供全面性的指導，就如同一本安裝之後的使用手冊。本文中以大量的 shell 指令範例來涵蓋系統管理的各個面向。包括的主題有： Debian 系統的基本概念、系統安裝提示、 Debian 套件管理、 Debian 中的 Linux 核心、系統微調、 gateway 建立、文字編輯器、 CVS 、程式設計、並為非開發者說明 GnuPG 等，每個主題都包括基礎指南、實用技巧和其它相關的資訊。
