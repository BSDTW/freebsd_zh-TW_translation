A 國安局支出 B 社會福利支出 C 社區發展支出 D 退休撫卹支出

A 一九四八年 B 一九五八年 C 一九六八年 D 一九七八年

A 耕者有其田 B 漲價歸公 C 發達國家資本 D 均富政策

A 鴉片戰爭 B 英法聯軍戰爭 C 中法戰爭 D 甲午戰爭

A 濟弱扶傾 B 修身 C 繼承道統 D 排滿

A 韓戰 B 越戰 C 八二三砲戰 D 以阿之戰

A 人民之保障書 B 權利之請願書 C 人民之請願書 D 權利之切結書

A 洛克 B 孟德斯鳩 C 漢彌爾頓 D 林肯

A 第九任總統 B 第十任總統 C 第十一任總統 D 第十二任總統

A 全體國民 B 全體公民 C 國家元首 D 全體民意代表

A 行政權 B 立法權 C 司法權 D 監察權

A 第六條 B 第七條 C 第八條 D 第九條

A 三日內 B 七日內 C 十日內 D 十五日內

A 十二小時 B 二十四小時 C 三十六小時 D 四十八小時

A 三個月 B 六個月 C 九個月 D 一年

A 立法院 B 司法院 C 監察院 D 國民大會

A 由總統召集 B 行政院院長召集 C 立法院院長召集 D 自行集會

A 立法院 B 行政院 C 考試院 D 監察院

A 二十歲 B 二十一歲 C 二十二歲 D 二十三歲

A 行政院院長 B 立法院院長 C 大法官 D 監察院院長

A 三十日內 B 四十日內 C 五十日內 D 六十日內

A 第八任 B 第九任 C 第十任 D 第十一任

A 三十八人 B 三十九人 C 四十人 D 四十一人

A 不編入於中央政府總預算案 B 不得刪減 C 不得加註意見 D 免送立法院審議
