﹝C﹞1.小過卦﹝﹞：「震上艮下，上動下止」其乃指：

       A 馴牛馬以致遠之象  B 舟楫之象  C 杵臼舂米

         之象  D 平息干戈之象

﹝D﹞2.鹽鐵論「貧富」一文中「食湖池，管山海，芻蕘不

       能與之爭澤，商賈不能與之爭利。」「芻蕘者」意

       指：

       A 農夫  B漁夫  C 工人  D樵夫

﹝D﹞3.荀子天論「君子不為小人匈匈也輟行。」「匈匈」意

       指：

       A 逆行  B 兇惡  C急促  D喧嘩

﹝B﹞4.漢書藝文志諸子略「敬順昊天，歷象日月星晨，敬

       授天時」係指：

       A 道家  B 陰陽家  C 農家  D儒家

﹝C﹞5.進學解一文中「沉浸濃郁，含英咀華。」句中「沉

       浸濃郁」其意指：

         之中  D義理正而詞藻美

       名，此句之係何人所言：

       A 子貢  B 子思  C 子夏  D 孟子

﹝D﹞7.定法一文中「術者，因任而授官，循名而責實，操

       生殺之柄，課群臣之能者也。」「課」意指：

       A 執行  B 授權  C 掌握  D 考核

﹝D﹞8.國有四維：一曰禮，二曰義，三曰廉，四曰恥。所

﹝C﹞9.阿房宮賦「盤盤焉，囷囷焉，蜂房水渦，矗不知幾

       千萬落。」「囷囷」意指：

       A 光禿貌  B 曲折貌  C 迴旋貌  D 直聳貌

       寫在發文者之前，並書寫全銜  C 公文在發文時應

       記國曆年月日及編列發文字號  D 機關名稱應寫全

       銜
